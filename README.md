# SmartCalc Universe

## Overview
SmartCalc Universe is a web-based application designed to help users perform various calculations such as unit conversions, factorials, summations, averages, income tax calculations, and payroll processing. It features an intuitive interface and is designed to be accessible and mobile-friendly.

## Features
- **Unit Conversion**: Convert temperatures and distances.
- **Factorial Calculator**: Calculate the factorial of a given number using a loop.
- **Sum and Average**: Find the sum and average of the first N natural numbers.
- **Income Tax Calculator**: Compute the tax based on progressive tax brackets.
- **Payroll Calculator**: Add employee details and calculate gross, deductions, and net pay.

## Technologies Used
- **HTML** for structure.
- **CSS** for styling.
- **JavaScript** for functionality.
